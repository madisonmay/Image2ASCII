Image2ascii

An image to ascii converter

Download this project as a .zip file Download this project as a tar.gz file

An Olin College Software Design Project

By Mitchell Kwock, Madison May, Jacob Kingery, and Bill Warner

Checking the code is as simple as: 

$ git clone https://github.com/madisonmay13/Image2ASCII.git

or 

$ git fork https://github.com/madisonmay13/Image2ASCII.git

What we set out to do

Image2Ascii is a project designed to push the limits of algorithmic ASCII art generation. Image2Ascii will eventually support both a real time ASCII conversion engine using greyscale comparison and a slower but more detailed character conversion method.

Check out a sample

If you'd like to take a look at the result, its as easy as git cloning the repo or downloading the tarball, then executing the following in the root directory of the project:

$ python terminal_movie.py

You'll have to install a few dependencies if you don't have them already installed. Currently, this project requires gimp, ffmpeg, mencoder, scikit, and cv.
